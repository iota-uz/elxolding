# Placeload

Placeloads allow you to mock the loading state of your components.

<Primary />

## Props

<Controls />

## Variants

<br />

### Text variant

Placeloads can be used to mock a block of text.

<div className="border border-muted-200 dark:border-muted-800 bg-white dark:bg-slate-900 p-6 rounded-sm">
  <Story of={PlaceloadStories.TextOne} />
  <Story of={PlaceloadStories.TextTwo} />
  <Story of={PlaceloadStories.TextThree} />
</div>

<br />

### Shape variant

Placeloads can be used to mock HTML elements.

<div className="flex gap-2 border border-muted-200 dark:border-muted-800 bg-white dark:bg-slate-900 p-6 rounded-sm">
  <Story of={PlaceloadStories.ShapeOne} />
  <Story of={PlaceloadStories.ShapeTwo} />
  <Story of={PlaceloadStories.ShapeThree} />
  <Story of={PlaceloadStories.ShapeFour} />
  <Story of={PlaceloadStories.ShapeFive} />
</div>

<br />

### Custom composition

Placeloads can be assembled together to create more complex elements.

<div className="flex gap-2 border border-muted-200 dark:border-muted-800 bg-white dark:bg-slate-900 p-6 rounded-sm">
  <div className="flex items-center gap-2">
    <Story of={PlaceloadStories.ShapeFour} />
    <div>
      <Story of={PlaceloadStories.TextOne} />
      <Story of={PlaceloadStories.TextTwo} />
      <Story of={PlaceloadStories.TextThree} />
    </div>
  </div>
</div>
